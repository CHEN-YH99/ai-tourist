### 数据模型
```
User (8字段, 2索引)
  ├─ email (唯一)
  ├─ password (加密)
  ├─ username
  ├─ avatar
  ├─ preferences
  └─ contactInfo

Conversation (4字段, 2索引)
  ├─ userId (可选)
  ├─ messages (数组)
  ├─ title
  └─ timestamps

Itinerary (8字段, 3索引)
  ├─ userId
  ├─ destination
  ├─ days (1-30)
  ├─ budget (>= 0)
  ├─ preferences
  ├─ content (DayPlan数组)
  └─ timestamps

Destination (12字段, 4索引)
  ├─ name (唯一)
  ├─ region, country
  ├─ type (数组)
  ├─ description
  ├─ images
  ├─ attractions
  ├─ averageBudget (min <= max)
  ├─ popularity (0-100)
  └─ timestamps

Collection (4字段, 3索引)
  ├─ userId
  ├─ itemId
  ├─ itemType (itinerary|conversation)
  └─ createdAt
```
