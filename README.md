# 🌍 AI旅游攻略助手

一个基于 Vue 3 + TypeScript 的智能旅游问答网站。

## 功能特点

- 💬 AI智能问答 - 实时解答旅游相关问题
- 🗺️ 个性化攻略推荐
- 🎨 现代化UI设计
- 📱 响应式布局

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

## 技术栈

- Vue 3
- TypeScript
- Vue Router
- Vite
